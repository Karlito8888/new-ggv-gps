# MyGGV-GPS - Brief

## Contexte
PWA de navigation type GPS pour un complexe résidentiel privé, nommée MyGGV-GPS.

## Objectif
Créer une application web progressive qui guide les visiteurs depuis leur position actuelle vers un emplacement spécifique (block et lot) dans l'enceinte, avec confirmation d'arrivée et options de navigation secondaire.

## Contraintes techniques
- Utiliser uniquement des outils 100% libres et open source
- Préférer MapLibre GL JS (uniquement) et OpenLayers pour les cartes
- Pas de TypeScript
- Utiliser Tailwind uniquement avec DaisyUI et shadcn/ui
- Interface en anglais

Trois mots d'ordre : - DRY
                     - KISS
                     - Minimaliste
